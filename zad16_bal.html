1. Opis znacznika DIV (jaki to element, składnia →w ramce, zastosowanie, opis treotyczny, atrybuty→ tylko wymień, opisz tylko ALIGN)

    • Element: Blokowy element kontenerowy Składnia: HTML Zastosowanie: Definiowanie sekcji strony Tworzenie układu strony Grupowanie elementów dla celów stylizacji Umieszczanie elementów JavaScript Opis teoretyczny: Znacznik div służy do definiowania obszaru blokowego w strukturze strony HTML. Jest elementem uniwersalnym, nie mającym domyślnego znaczenia semantycznego. Atrybuty: align: Wyrównanie zawartości (lewo, prawo, środek) class: Nazwa klasy CSS id: Unikatowy identyfikator elementu style: Właściwości CSS title: Podpowiedź tekstowa Opis atrybutu align: Atrybut align służy do wyrównania zawartości elementu div w poziomie. Możliwe wartości to: left: Wyrównanie do lewej right: Wyrównanie do prawej center: Wyrównanie do środka

  2. Opis znacznika SPAN (jaki to element, składnia →w ramce, zastosowanie, opis treotyczny).

    • Element: Element liniowy Składnia: HTML Zastosowanie: Formatowanie fragmentów tekstu Dodawanie stylów do tekstu Grupowanie tekstu dla celów JavaScript Opis teoretyczny: Znacznik span służy do definiowania fragmentu tekstu o określonych właściwościach. Jest elementem liniowym, co oznacza, że nie powoduje łamania linii tekstu. Atrybuty: class: Nazwa klasy CSS id: Unikatowy identyfikator elementu style: Właściwości CSS title: Podpowiedź tekstowa

  3. Różnica między znacznikami DIV i SPAN.

    • Podstawowa różnica: div jest elementem blokowym, a span elementem liniowym. Inne różnice: div może zawierać inne elementy blokowe i liniowe, a span może zawierać tylko elementy liniowe. div domyślnie zajmuje całą szerokość dostępnego miejsca, a span dopasowuje się do szerokości zawartości. div często używany jest do tworzenia układu strony, a span do formatowania tekstu.
